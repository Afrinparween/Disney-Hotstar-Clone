# Disney-Hotstar-Clone

A responsive **Disney+ Hotstar Clone** built using **HTML, CSS, and JavaScript**. This project recreates the Disney+ landing page with a modern UI, interactive carousel, hover video previews, and movie sections.

## 📌 Features

- 🎥 Auto-sliding image carousel
- 📺 Movie and TV show categories
- 🔍 Search bar
- 🎬 Video preview on hover
- 📱 Responsive design
- 🎞️ Horizontal scrolling movie cards
- ❤️ Add to Watchlist button
- 🎨 Clean and modern Disney+ inspired interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
